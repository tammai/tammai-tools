# Release Notes — v0.2.0 (2026-06-22)

## What's new

### Marketplace support

`tammai-tools` is now listed in the Claude Cowork marketplace. A new `.claude-plugin/marketplace.json` file provides the metadata Claude Cowork needs to display and distribute the plugin:

- **Categories**: Productivity, Presentations, Education
- **Tags**: slides, workshop, html, bigin, dark-theme
- **Pricing**: Free
- **Icon**: `skills/workshop-slides/assets/favicon.ico`

### Horizontal navigator (workshop-slides)

The slide navigator in generated decks has been redesigned from a vertical column to a horizontal bar pinned to the bottom-right corner. Chevron icons now point left/right instead of up/down, matching the horizontal slide direction.

## Upgrading

No action required. Reinstall the plugin from the `.plugin` bundle to pick up the new version.
